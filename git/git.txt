git config --global user.name SergeyRushev
git config --global user.email rushev84@yandex.ru

git init

Добавить все файлы внутри проекта:
git add .
git add css/**/*.html (все дочерние папки с html)
git add !css/style.css (все папки из css кроме style.css)

Посмотреть статус файлов:
git status

Удалить из ожидания:
Один файл
git rm .idea/workspace.xml --cached
Целую папку
git rm -r .idea --cached
Подробнее об удалении из ожидания: https://qna.habr.com/q/429910

Коммит:
git commit -m "comment"

Вся цепочка изменений:
git log
git log --oneline

Перейти на определенный коммит:
git checkout <id коммита>
git checkout master

Отменить (без удаления) определенный коммит:
git revert <id коммита>
Удаление коммита (файлы остаются без изменений):
git reset <id коммита>
Удаление коммита и возвращение файлов в докоммитный вид:
git reset <id коммита> --hard

Создать новую ветку:
git branch <имя ветки>
Удалить ветку:
git branch -D test
Перейти на другую ветку:
git checkout <имя ветки>
Создать ветку и сразу перейти на неё:
git checkout -b <имя ветки>
Вывести все ветки проекта:
git branch -a
Слияние веток:
git merge <имя ветки>

Подключиться к удалённому репозиторию:
git remote add origin https://github.com/rushev84/inet_test.git
Проверить подключение (если пишет origin, то всё ок):
git remote
Забросить файлы в удалённый репозиторий:
git push -u origin master

Клонирование:
git clone https://github.com/rushev84/inet_test.git

Утянуть:
git pull

Когда не добавляется в удалённый репозиторий:
git push origin HEAD:master
