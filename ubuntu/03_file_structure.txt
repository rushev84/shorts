bin - стандартные утилиты и основные исполняемые файлы
boot - ядро, конфиг для загрузки ядра
dev - файлы устройств
etc - конфиги служб, вспомогательных систем
home - домашняя директория пользователей
lib - библиотеки и модули ядра
lost+found - аналог корзины (хранит битые файлы, потерянные)
media, mnt - примонтированные диски
opt - установленные программы большого размера
proc - псевдофайловая система, кот. предоставляет информацию о процессах
swapfile - файл подкачки
tmp - временные файлы
usr - пользовательские программы, документация и т.д.
var - меняющиеся файлы (кэши, логи, файлы БД)
